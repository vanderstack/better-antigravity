# Skill Registry

This directory contains a suite of specialized skills for agents working within the Antigravity environment. Use this registry to find the appropriate mental model and guidance for your task.

| Skill Filename | Description | Context of Value |
| :--- | :--- | :--- |
| **[sdk-onboarding.md](./sdk-onboarding.md)** | Protocol-level SDK integration and diagnostic mastery. | Essential for deep backend tasks, identifying internal RPC signals, and working with "Cortex" events. |
| **[skill-planning.md](./skill-planning.md)** | Meta-skill for designing new guidance artifacts. | Use this when planning or writing new skills to ensure they meet the project's standards of excellence and architectural grounding. |
| **[telegram-bridge.md](./telegram-bridge.md)** | Messaging and attachment delivery via Telegram. | The primary way to relay status, logs, and artifacts to human users via the high-reliability outbox queue. |

## 🚀 How to use these Skills

1.  **Read the Registry first**: Identify which path (UI vs. Protocol) aligns with your current objective.
2.  **Refer to the linked Skill**: Each file provides concrete instructions, "Think-First" strategies, and technical deep-dives.
3.  **Follow the Mental Model**: These skills are designed to calibrated your reasoning, not just provide a list of commands.
